## The Consensus Signal

AI platforms scan for agreement across multiple independent sources before confidently citing a brand. Consistent positioning across Reddit, YouTube, industry publications, review sites, and your website dramatically increases citation likelihood.

## Platform-Specific Citation Behavior

Only 11% of domains are cited by BOTH ChatGPT and Perplexity. Each LLM has distinct patterns:

**ChatGPT:**
- Heavily favors Wikipedia and authoritative domains
- Parametric knowledge (training data) + optional web search
- OAI-SearchBot builds search index from HTML content
- Favors well-established brands with Wikipedia presence

**Perplexity:**
- Real-time RAG for every query — searches live web
- Breaks queries into sub-queries, searches independently
- Favors fresh, recently updated content
- Reddit content surfaces frequently
- Transparent numbered citations

**Google AI Overviews / Gemini:**
- Leverages existing Google Search index
- Benefits from traditional SEO + structured data
- Cross-references multiple sources

**Claude:**
- Claude-SearchBot indexes content for search
- Claude-User fetches on demand during conversations
- Reads raw HTML/text content

## What LLMs Actually Read

All LLMs share these patterns:
- Read **visible page text** (NOT hidden JSON-LD schema)
- Parse **heading hierarchy** (H1-H6) for structure
- Extract **lists, tables, structured content** preferentially
- Use **meta descriptions** and **title tags** as context signals
- Benefit from **clean semantic HTML**
- Do **NOT execute JavaScript** — SSR/static content only
- JSON-LD treated as regular text, not machine-readable data (SearchVIU 2025)

## Citation Accuracy Warning

50-90% of LLM responses are not fully supported by cited sources (Nature Communications study). Being cited doesn't guarantee accurate representation.

## Actionable Citation Checklist

1. Lead with a direct answer (answer-first, then expand)
2. Embed 3-5 specific, citable numbers per page (stats, percentages, benchmarks)
3. Use 2-3 prominent pull quotes with key findings
4. Structure content modularly (H2/H3 sections that stand alone)
5. Include FAQ sections with clear Q&A format
6. Maintain 1,500+ word depth on key pages
7. Cite your own sources (references signal credibility)
8. Update high-value pages every 60-90 days
9. Build cross-platform presence (Reddit, G2, YouTube, industry pubs)
10. Ensure robots.txt allows AI crawlers
11. Use semantic HTML (headings for structure, lists for steps, tables for comparisons)
12. Keep key facts in visible text (not in images, PDFs, or JS-rendered content)
